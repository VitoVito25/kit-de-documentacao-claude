# Prompt — Novo Projeto a partir deste Kit

Cole este prompt no agente ao iniciar um projeto novo baseado neste template.

---

Vou configurar este kit de documentação para o novo projeto. Preciso de 4 informações:

1. **Nome do projeto** — usado em todos os frontmatters
2. **Stack técnica** — linguagem, framework principal, gerenciador de estado, test runner, bundler (ex: "React + TypeScript, Zustand, Vitest, Vite")
3. **Estrutura de pastas** — raiz do projeto e caminho da pasta de utils/componentes compartilhados (ex: `src/`, `src/components/shared/`)
4. **Padrão arquitetural** — como o código é organizado (ex: container/presentational, feature-based, MVC, clean architecture)

Com essas respostas vou:
- Substituir `[NOME DO PROJETO]` em todos os frontmatters
- Preencher `YYYY-MM-DD` com a data atual em todos os arquivos
- Preencher `[RAIZ DO PROJETO]`, `[pasta de componentes compartilhados]` e `[ComponenteA].tsx` no `dry-refactoring.md`
- Adaptar as seções de stack em `coding-conventions.md`, `ai-best-practices.md` e `performance-audit.md` seguindo as instruções `[COMO PREENCHER]` de cada arquivo
- Adaptar o padrão arquitetural em `separation-of-concerns.md`
- Remover os blocos `> **[COMO PREENCHER]**` após preencher cada seção
